# **App Name**: Artifolio

## Core Features:

- Profile Showcase: Display personal information including a bio, headshot, and contact details.
- Skills Overview: Present a visual summary of skills, possibly with proficiency levels.
- Project Gallery: Exhibit projects with images, descriptions, and links to live sites or repositories.
- Experience Timeline: Showcase achievements and experiences in a chronological order.
- Feedback Carousel: Rotate through testimonials or feedback received.
- Link Aggregator: Provide links to external profiles like LinkedIn, GitHub, and CodePen.
- AI-Powered Resume Tailor: Upload job description document to produce a bullet point list that makes it easy for users to see which elements from their project gallery to focus on when filling in their resume, and explains why it has suggested these projects; this is provided as a tool that helps users quickly write relevant materials for each individual job.

## Style Guidelines:

- Primary color: Deep navy blue (#2E3192) for a professional yet creative feel.
- Background color: Off-white (#F0F0F5) to provide a clean and modern canvas.
- Accent color: Electric purple (#BE0AFF) to highlight key interactive elements and calls to action.
- Headline font: 'Belleza' (humanist sans-serif), to establish an artistic and fashionable feel. Body Font: 'Alegreya' (humanist serif)
- Use a set of simple, consistent icons for skills and contact links.
- Implement a fluid, responsive grid layout that adapts well to various screen sizes, prioritizing readability.
- Incorporate subtle parallax scrolling effects and tasteful transitions between sections to add depth and engagement without overwhelming the user.
